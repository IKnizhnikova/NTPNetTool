Nested Time Petri Nets tools (v.0.0.1 prealpha)

The prototype of modelling and analysis of Nested Petri Nets tools.
Workin features:

  - modelling Time Nested Petri nets
  - CTL formulae editor
  - reachability essential integer states graph algorithm
  - CTL model checking tools

jar:
jar/ - compiled NTPNetTool modules 
  drop them into you Eclipse plugin folder
    
src:
src/ - NTPNetTools modules sources
  create a workspace in Eclipse Modelling Tools
  import these projects
  compile
